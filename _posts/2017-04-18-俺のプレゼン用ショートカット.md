プレゼンする際によく使用するMacのショートカット集です。  
末尾に修飾キーの読み方も載せてます。  
  
## ディスプレイ周り  
### ミラーリング  
  
`⌘` + `F1`  
  
### Dock非表示  
  
`⌥` + `⌘` + `d`  
  
### アプリ切り替え  
  
`⌘` + `tab`  
  
### アプリ内ウィンドウ切り替え  
  
`⌘` + `F1(+fn)`  
  
  
## ブラウザ周り（Chrome）  
  
### フルスクリーン  
  
`⌃` + `⌘` + `f`  
  
### ブックマークバーを隠す  
  
`⇧` + `⌘` + `b`  
  
## 備考：修飾キーの意味  
  
- `⌘` : command  
- `⌥` : option (alt)  
- `⇧` : shift  
- `⌃` : control
