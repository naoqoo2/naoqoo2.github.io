# サンドボックスでネットワークを許可する

`gh` コマンドで GitHub の PR を見たい時、サンドボックスのネットワークが遮断されていると失敗します。（以前は不要だったのにいつからかセキュリティ周り厳しくなってました）

下記を追記すればOK！

```bash
vi ~/.codex/config.toml
```

```toml
[sandbox_workspace_write]
network_access = true
```

- `sandbox_workspace_write` セクションで `network_access = true` にすると、サンドボックスから外部ネットワークへ出られるようになります。
- これで `gh pr view` や `gh pr checkout` がサンドボックス内で動作するようになります。
